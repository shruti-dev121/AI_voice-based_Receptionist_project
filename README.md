 ## Overview
 The Hospital Voice Chatbot is a web-based intelligent assistant designed to automate patient
 interaction tasks in hospitals. It enables voice-based data collection, form filling, appointment queries,
 billing assistance, and more. The chatbot helps streamline workflow and reduce manual effort for
 hospital receptionists.
 ## Features- Voice-based input for patient details (Name, DOB, Address, etc.)- Real-time speech-to-text conversion- Form auto-filling via voice- Storage of data into Excel or MySQL- Easy-to-use web interface- Extendable backend built with Python/Flask- Fast API responses for patient registration and queries
 ## Tech Stack
 ### Frontend- HTML- CSS- JavaScript
 ### Backend- Python- Flask Framework
 ### Database- Excel (initial storage)- MySQL (optional structured storage
